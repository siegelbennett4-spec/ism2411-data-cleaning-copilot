# ism2411-data-cleaning-copilot

Small data-cleaning demo for ISM2411. The `src/data_cleaning.py` script
shows a concise cleaning pipeline for a messy sales CSV and writes a
cleaned CSV to `data/processed/sales_data_clean.csv`.

Quick start:

1. Install dependencies (if needed):

```bash
python -m pip install pandas
```

2. Run the cleaner:

```bash
python src/data_cleaning.py
```

3. Inspect the output at `data/processed/sales_data_clean.csv`.

The repository includes a sample messy `data/raw/sales_data_raw.csv` so
you can run the pipeline immediately.
